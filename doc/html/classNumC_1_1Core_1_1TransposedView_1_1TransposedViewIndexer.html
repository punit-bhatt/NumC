Calculates the memory index corresponding to the index parameter for the array/view.

Parameters
indexArray/view index.
Returns
Memory index.

Overridden function.

Note
The logic is pretty simple. Find the coordinates corresponding to the index for the transposed view. Multiply these by the reordered data array strides.
Example,
(2, 3) - strides(3, 1)
[0, 1, 2],
[3, 4, 5].
Transposed view -
(3, 2) - strides(2, 1)
[0, 3],
[1, 4],
[2, 5]
Transposed view index 1 would be element 3. Calculate coordinates for this - 0th row, 1st column. Multiply by these by the reordered strides - 0 * 1 + 1 * 3 = 3

Implements NumC::Core::MemoryIndexer.
