Detailed Description

template<typename T>
class NumC::Core::ReshapedView< T >

Reshaped view of the array object.

A reshape can potentially change the number of dimensions and the dimensions themselves. This would lead to change in the strides as well. However, it cannot change the total number of units.

Construct a new Reshaped View object.

Note
Reshaped View can have different shape/dimensions along with the overall number of dimensions. However, the number of elements/units can't differ from that of the array being reshaped.
Warning
Reshaping of other type of Views not supported. Reshaping is only allowed upon memory containers or other Reshaped Views.
Parameters
arrayPointer to the array object.
newShapeThe new shape/dims to reshape the array to. One shape dimension can be -1. In this case, the value is inferred from the length of the array and remaining dimensions.

Note
